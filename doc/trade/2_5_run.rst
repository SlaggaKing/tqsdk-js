运行
=======================================

恭喜，马上就可以正式运行任务下单了。

方式一、在天勤客户端中
------------------------------------------------------

右击刚刚添加的板块，右键菜单中选择刷新。

方式二、在 Chrome 浏览器中打开
------------------------------------------------------
刷新页面，http://taide.tq18.cn/trader/trader_user.html。

单击买开或者卖开按钮，即可开始运行下单任务。试试吧。

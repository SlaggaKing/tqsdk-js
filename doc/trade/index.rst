第一章 开始上手
----------

在这一章中通过一个完整的示例，了解如何使用 javascript 编写你想要交易程序。

1. 检查环境
~~~~~~~~~~

在开始真正编写代码前，先来检查以下你的电脑环境，确保达到以下要求：

+ 首先当然是需要安装天勤客户端，下载地址 `天勤客户端 <http://upgrade.tq18.cn/efng/tqsetup_0.7.0.0.exe/>`_ 。
+ 选择一款文本编辑器
    - sublime
    - vscode
    - nodepad++
    - 等等
+ Chrome 浏览器，版本，下载地址 `Chrome 浏览器 <https://www.google.com/chrome/browser/desktop/index.html/>`_ 。

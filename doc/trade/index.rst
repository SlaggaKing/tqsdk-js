----------------------------------------------------------------------

1. 检查环境
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

在开始真正编写代码前，先来检查以下你的电脑环境，确保达到以下要求：

+ 首先当然是需要安装天勤客户端， `天勤客户端下载地址`_ 。
+ 选择一款文本编辑器
    - sublime
    - vscode
    - nodepad++
    - 等等
+ Chrome 浏览器， `Chrome 浏览器下载地址`_ 。

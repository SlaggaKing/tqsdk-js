天勤技术指标系统
===================================
本软件支持用户以三种方式创建自定义技术指标。这三种方式创建的技术指标在使用时是完全一致的。

* 用 javascript 编写技术指标
* 转换其它语言格式的技术指标到天勤的语法格式
* 通过websocket接口，外挂指标计算扩展模块

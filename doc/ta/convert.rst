从其它格式的技术指标转换
=======================================
#. 打开天勤终端, 点击 [指标编辑], 弹出 [指标编辑] 对话框
#. 点击 [新建指标], 输入 [指标名称], 并选择 [文华脚本语言]
#. 将指标代码黏贴到代码编辑区
#. 将指标参数填入 参数列表
#. 点击 [保存代码并更新指标]
#. 可以在图表中正常使用新加入的指标

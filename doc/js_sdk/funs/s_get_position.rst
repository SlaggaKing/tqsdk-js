方法 C.GET_POSITION
==================================

获取持仓信息

.. js:function:: GET_POSITION()

   :returns: 返回账户全部持仓对象。

数据源
----------------------------------

+------------------------+------------+--------------+
| from                   | id         | default      |
+========================+============+==============+
| Ctx.LATEST_DATA        | instrument | CFFEX.TF1803 |
+------------------------+------------+--------------+
| Ctx.LAST_UPDATED_DATA  | volume     | 3            |
+------------------------+------------+--------------+

示例
----------------------------------

.. code-block:: javascript

    function * Task(C){
        ...
        var order = C.GET_POSITION();
        ...
    }
